# Rock-Paper-Scissors Game with Authentication

This is a web-based **Rock-Paper-Scissors** game that includes **user registration and login** functionality using HTML, CSS, and JavaScript. The project demonstrates:

- User registration with validation (email & strong password)
- Login authentication using `localStorage`
- Welcome screen displaying the user's name
- Single-player mode (vs CPU) and two-player mode (vs friend)
- Interactive game interface with move selection and winner determination
- Email auto-completion feature
